## 📂 Projects

### [01 · Tipping Behavior — Regression Analysis](./01_tipping-behavior/)
> *Statistical Analysis · Python · OLS Regression · Hypothesis Testing*

End-to-end analysis of 244 restaurant tables across 7 days. Built three OLS regression models to identify the true drivers of tip amounts — controlling for total bill, party size, day, and shift. Includes t-tests, Cohen's d effect sizes, correlation analysis, and an executive summary deck for non-technical stakeholders.

**Key finding:** Bill size is the primary driver of tip amount. Guests tip at roughly the same *rate* regardless of time, day, or demographics — what changes the absolute tip is how much they spend.

`Python` `Statsmodels OLS` `Seaborn` `Pandas` `Hypothesis Testing` `SciPy`

### [02 · Eastville Real Estate Analysis](./02_real-estate-analysis/)
> *Exploratory Data Analysis · Python · Correlation · Effect Size*

Comprehensive EDA of 108 Eastville homes examining what drives residential property prices. Combines single-variable analysis, two-variable comparisons, statistical significance testing, and practical effect size measurement (Cohen's d).

**Key finding:** Square footage is the strongest price driver (r = 0.772). Ranch-style homes command a significant premium over Cape Cod and Two-story homes; the Cape Cod vs. Two-story price difference is not statistically significant (p = 0.65).

`Python` `SciPy` `Seaborn` `Matplotlib` `Correlation Analysis` `Cohen's d` `Feature Engineering`

### [05 · Causal Alignment Prompting (CAP) — LLM Synthetic Survey Augmentation](./05_CAP-llm-augmentation/)
> *Causal Inference · Python · RCT Design · LLM Augmentation · Hypothesis Testing*

Tested whether large language models can act as synthetic survey respondents to augment a small policy-framing RCT without distorting causal estimates. Designed and implemented **Causal Alignment Prompting (CAP)** — a preregistered validation framework — on a two-arm gain vs. loss experiment (N = 253). The augmentation failed equivalence criteria, demonstrating why structured validation protocols matter before using AI-generated data for inference.
