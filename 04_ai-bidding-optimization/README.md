## Overview

Collaborated with Techbridge to develop machine learning models optimizing bidding behavior in a real-world procurement context. Built an end-to-end pipeline from raw data to validated models, with a focus on data integrity and reproducibility.

## Responsibilities

- Data ingestion, cleaning, and validation across large-scale procurement datasets
- Feature engineering for bidding behavior modeling
- Model training, tuning, and validation in Python
- Version control and collaborative development via GitHub
